# Mercia's Quill Translations

This repository is used for managing all localised translations for Mercia's Quill via [Crowin](https://crowdin.com). For any questions, concerns, or clarification, please join the Mercia's Quill Discord server [support server](https://merciasquill.com/support).

## Workflow 

- All original source files are listed in the /source/ directory. This should not be altered for any tranlsations, only to add new keys, or correct typos in the original english text.
- All translated files are listed in the /locales/id/ directory. While these files can be directly altered, it is recommended to translate new keys directly from [Crowdin](https://crowdin.com/project/merciasquill).
